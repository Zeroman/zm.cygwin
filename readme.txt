1. 文件加压到指定目录后，运行install.bat, 等待安装完毕。安装时间受网速影响，至少要几分钟
2. 安装后会弹出字体安装窗口，点击安装
3. 运行start.bat
4. 进入shell命令后，运行 git clone https://github.com/Zeroman/zm.config.git
5. cd zm.config, 执行./install.sh， 可能出现错误，执行3遍
6. 等待安装完毕，安装完毕输入v进入vim
7. windows的c盘和d盘在shell里对应的目录是/cygdrive/c和/cygdrive/d
8. gen_tags命令用来生成c和c++工程需要的tags文件
